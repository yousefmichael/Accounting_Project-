📌 Project Overview
The Integrated Accounting System is a lightweight, responsive, and bilingual (Arabic & English) web-based financial management application. Built natively with HTML, CSS, and Vanilla JavaScript, it is designed to help local businesses or accounting students effectively manage core financial operations, record transactions, and automatically generate essential financial statements without the need for complex backend infrastructure.

✨ Key Features
📊 Interactive Dashboard (لوحة التحكم):
Provides a high-level overview of account balances, recent entry activities, and a quick financial summary.

🗂️ Chart of Accounts (دليل الحسابات):
A centralized ledger for organizing, classifying, and managing all financial accounts systematically.

📝 Journal Entries (قيود اليومية):
A robust interface for recording double-entry transactions (Debit/Credit), ensuring the fundamental accounting equation is constantly maintained.

⚖️ Trial Balance (ميزان المراجعة):
Automatically aggregates debit and credit balances to verify mathematical accuracy before creating financial statements.

💰 Income Statement (قائمة الدخل):
Dynamically calculates and presents revenues and expenses to determine net income or loss over a period.

🏦 Balance Sheet (الميزانية العمومية):
A real-time report detailing assets, liabilities, and equity to reflect the current financial position.

💾 Saved Entries (القيود المحفوظة):
A continuous log of historically recorded transactions for easy tracking, auditing, and review.
📈 Export to Excel: Built-in capability leveraging the SheetJS library to easily export data to .xlsx formats for off-device auditing or backup.
🖨️ Print-Ready: Optimized and clean interface specifically tailored for generating printed reports directly from the browser.

🛠️ Tech Stack
Frontend: HTML5, Modern CSS Variables & Gradients, Vanilla JavaScript (ES6+).
Libraries: SheetJS for Excel data exports.
Styling: Custom "Glassmorphism" UI with responsive grid layouts.
Typography: Google Fonts (Cairo for readable Arabic script and JetBrains Mono for clear numbers/data).
Storage: LocalStorage (for client-side data persistence).
